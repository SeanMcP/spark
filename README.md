# Spark

## Arguments

| Argument    | Shortcut | Description            |
| ----------- | -------- | ---------------------- |
| `--class`   | `-C`     | create class component |
| `--connect` | `-c`     | connect to Redux       |

Run `spark --help` to view arguments.

## Configuration

`.sparkrc.json`:

| Property  | Value   | Default |
| --------- | ------- | ------- |
| output    | string  | `src`   |
| filetype  | string  | `jsx`   |
| proptypes | boolean | `true`  |
